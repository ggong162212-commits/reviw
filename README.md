# 리뷰 파워블로거 변환기 ✨

별점·장단점·총평만 입력하면, 그 내용을 분석해 **파워블로거 감성 리뷰**로 바꿔주는 웹앱입니다.
에이블리·쿠팡·무신사·네이버·올리브영 등 쇼핑몰 리뷰에 바로 붙여넣기 좋게 만들었어요.

## 쓰는 법

1. `index.html` 파일을 더블클릭해서 브라우저로 열어요. (모바일에서도 OK)
2. 입력합니다.
   - **상품명 / 카테고리**
   - **별점** (별 클릭)
   - **👍 장점** — 줄바꿈 또는 `,` 로 여러 개
   - **👎 단점** — 없으면 비워둬도 됨
   - **📝 총평** — 한 줄만 적어도 됨
3. **플랫폼**(에이블리/쿠팡 등)과 **말투**(감성/친근/정보/솔직)를 고릅니다.
4. **✨ 파워블로거 리뷰로 변환** 버튼을 누릅니다.
5. **🔄 다시** 로 다른 버전을 뽑고, **📋 복사** 로 복사해서 리뷰에 붙여넣어요.

## 특징

- **설치·로그인·인터넷 불필요** — 파일 하나만 있으면 됩니다.
- **개인정보 안전** — 입력 내용은 전송·저장되지 않고 100% 내 브라우저에서만 처리돼요.
- **4가지 말투** — 감성 충만 🌸 / 친근 발랄 😆 / 정보 꼼꼼 🔍 / 솔직 담백 ✌️
- 별점에 맞춰 마무리 멘트(강추/무난/신중)와 톤이 자동으로 달라집니다.
- 카테고리·플랫폼에 맞는 **해시태그 자동 생성**.
- 누를 때마다 문구가 살짝 바뀌어서 똑같은 리뷰가 안 나와요.

## 파일 구성

- `index.html` — 앱 전체 (HTML·CSS·JS 한 파일)

별점이 4점 이상이면 추천 톤, 3점이면 무난 톤, 그 이하면 솔직·신중 톤으로 자동 조정됩니다.
